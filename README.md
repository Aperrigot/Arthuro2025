# Arthuro2025

Le futur dans un ordinateur cassé

## Structure du projet

- `src/index.html` : page principale du site de vente de plantes.
- `src/css/style.css` : styles de la boutique.
- `src/js/main.js` : logique simple pour gérer le panier.

Ouvrez `src/index.html` dans un navigateur pour découvrir la boutique.
